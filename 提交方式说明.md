# 作业提交方式说明（降低参与门槛）

学生**不需要登录 Google** 即可提交作业，有以下三种方式。

---

## 方式一：下载 JSON 后自行发送（无需任何配置）

- 学生点击 **「Download JSON」**，浏览器会下载一个 `.json` 文件。
- 学生将该文件作为附件，通过自己的邮箱发送给老师。
- **优点**：无需配置、无需后端，学生不登录任何账号。
- **缺点**：需要学生多一步（打开邮箱、添加附件、发送）。

---

## 方式二：点击「Submit to teacher」直接发到老师邮箱（推荐）

学生点击 **「Submit to teacher」** 后，JSON 会通过后台接口**以邮件附件形式发送到老师邮箱**，学生无需登录。

### 老师需要做的配置（一次性）

1. **注册 Resend**  
   打开 [resend.com](https://resend.com) 注册账号，在 Dashboard 里创建 API Key（免费版约 100 封/天，足够作业提交）。

2. **部署到 Vercel 并配置环境变量**  
   在 Vercel 项目里添加以下环境变量：

   | 变量名 | 说明 | 示例 |
   |--------|------|------|
   | `RESEND_API_KEY` | Resend 的 API Key | `re_xxxxxx` |
   | `TEACHER_EMAIL` | 接收作业的邮箱 | `teacher@school.edu` |
   | `FROM_EMAIL`（可选） | 发件人显示，不填则用 Resend 默认 | `LandscapePro <onboarding@resend.dev>` |

3. **重新部署**  
   保存环境变量后，在 Vercel 里重新部署一次，使接口生效。

### 学生端

- 打开你部署的网页，做完设计后点击 **「Submit to teacher」**。
- 页面会显示「提交成功！已发送到老师邮箱」。
- 学生无需登录、无需填写邮箱，门槛最低。

### 技术说明

- 前端会把 JSON 通过 POST 发送到 **`/api/submit`**（与网站同源，例如 `https://你的项目.vercel.app/api/submit`）。
- 项目里的 **`api/submit.js`** 是 Vercel Serverless 函数，会调用 Resend 把 JSON 作为附件发到 `TEACHER_EMAIL`。

---

## 方式三：上传到 Google Drive（可选，需学生登录）

- 学生点击 **「Upload to Google Drive」**，需用 Google 账号登录后，文件会保存到您指定的 Drive 文件夹。
- **优点**：集中在一个文件夹里查看。
- **缺点**：必须登录 Google，参与门槛较高；仅适合仍希望使用 Drive 的情况。

---

## 总结建议

- **不强制使用 Google**：以 **「Download JSON」** 和 **「Submit to teacher」** 为主。
- **最低门槛**：配置好 Resend + Vercel 环境变量后，学生只需点 **「Submit to teacher」**，即可把 JSON 发到老师邮箱，无需登录、无需自己发邮件。

如有部署或环境变量问题，可参考项目中的 `VERCEL部署详细步骤.md`。
